# Assignment-01-Student-Admission-Eligibility-Checker
Description

This Python program collects student details and checks whether the student is eligible for admission based on their percentage.

Features
Accepts student details:
Name
Age
Degree
Percentage
Displays student information.
Checks admission eligibility:
Eligible if percentage is 60 or above.
Not Eligible if percentage is below 60.
Calculates:
Years remaining to reach age 30.
Percentage required to reach 100.

Python

Variables
Input and Output
Data Types (int, float, string)
Conditional Statements (if-else)
Arithmetic Operations

Student Details
